# GitHub Activity Tracker

A Python script that fetches your GitHub activity and sends a weekly email report.

## Features
- Fetches recent GitHub events via GitHub API
- Sends automated email report via Gmail
- Credentials stored securely using .env file

## Setup
1. Clone the repo
2. Install dependencies: `pip install requests python-dotenv`
3. Create a `.env` file with your Gmail app password:
   `APP_PASSWORD=your_app_password`
4. Run: `python tracker.py`
